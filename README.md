# Project
국내 숙박/휴가/레저/여행을 한 번에 예약 할 수있는 플랫폼 여기어때 를 모티브로 한 숙박 중계 플랫폼!
### 선정이유
- 모티브하게된 여기어때 페이지에서 이용되는 소셜 로그인 및 지도 기능
- 대실 및 숙박과 같은 여러가지 예약에 차이를 두는 기능
- 현재 위치를 기반으로 이용자에게 가장 가깝거나 값이 저렴한 상품들을 보여주는 기능

## Project기간
2022.10.31 ~ 2022.11.10
